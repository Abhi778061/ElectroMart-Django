\# ElectroMart – Django E-Commerce Website



ElectroMart is a full-stack e-commerce web application developed using Django.  

It provides complete online shopping functionality with product browsing, cart, checkout, orders, wishlist, and invoice generation.



---



\## Features



\- User authentication (Login \& Register)

\- Product listing with images

\- Product detail pages

\- Add to cart functionality

\- Wishlist feature

\- Checkout and order placement

\- Order history

\- Invoice (PDF) generation

\- Responsive UI using HTML \& CSS



---



\## Technologies Used



\- Python

\- Django

\- HTML

\- CSS

\- JavaScript

\- SQLite (Database)



---



\## Project Structure



\- `Quick\_cart/` – Main Django app

\- `templates/` – HTML templates

\- `media/` – Product images

\- `static/` – CSS and JS files

\- `manage.py` – Django project runner



---



\## How to Run the Project Locally



1\. Clone the repository  

2\. Create a virtual environment  

3\. Install dependencies  

4\. Run migrations  

5\. Start the server
